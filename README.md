# -GEO_HaT_Xenium_Submission
This repository contains raw and processed data, code, and figures used in the analysis of spatial transcriptomic differences between HaT and non-HaT colon samples.

## Contents
- **raw_data/**: Unaltered Xenium exports from 10x Genomics
- **processed_data/**: Aggregated counts, DESeq2 results, normalized matrices
- **code/**: R Markdown and scripts used for all figures and analysis
- **figures/**: Key images used in the manuscript

## Software and Dependencies
- R (>= 4.3)
- Seurat (v5)
- DESeq2
- ggplot2, ggrepel
